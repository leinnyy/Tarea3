Web Responsivo

Es un enfoque de diseño que permite que un sitio web se adapte a diferentes tamaños de pantalla y dispositivos. En la actualidad, esto ofrece una mejor experiencia al usuario.

Viewport

El viewport se refiere a la ventana visible de una página web en el navegador. Su tamaño puede variar dependiendo del dispositivo utilizado: smartphones, tablets, desktops, etc.

Cuando se trabaja con viewport se debe saber su importancia al implementarlo. Algunos puntos importantes son:

  1. Tamaño de la Página
  2. Adaptación de Estilos
  3. Optimización de la Experiencia del Usuario.

La configuración más utilizada del viewport se hace con la etiqueta de meta.

Las media queries son una herramienta fundamental en CSS para crear diseños web responsivos que se adaptan a diferentes tamaños de pantalla y resoluciones.

@media (max-width: 768px) { .container { max-width: 1000px; padding: 20px; } }
